Get Thunderbolt running locally in a few steps.

## Prerequisites

- [Bun](https://bun.sh) installed
- A supported AI provider API key (OpenAI, Anthropic, etc.)

## Installation

```bash
git clone https://github.com/thunderbird/thunderbolt.git
cd thunderbolt
bun install
```

## Development

```bash
bun dev
```

Visit `http://localhost:5173` to see Thunderbolt running.

## Configuration

Create a `.env` file in the project root:

```bash
cp .env.example .env
```

Add your API keys and configure your preferred AI models.
